Use of RxJS and Its Advantages

Angular's Reactive Nature

  • Angular is a framework developed by Google
  • "Reactivity" is one of the core principles
  • Angular relies on RxJS

What is RxJS?

  • Library for reactive programming
  • Enables developers to work with asynchronous data streams
  • Streams can represent events, HTTP requests, user interactions etc.

How Angular Uses RxJS?

  • RxJS observables and operators are used for:
    • event handling
    • data manipoation
    • asynchronous operations

  • Enables developers to work with asynchronous data streams
The Power of Angular's RxJS Integration
  • Streamlined Asynchronous Operations
    RxJS simplifies handling asynchronous tasks such as HTTP requests and timers
    
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

export class ExampleService {
  constructor(private http: HttpClient) {}

  // Method that performs an HTTP GET request using RxJS
  fetchData(): Observable {
    const apiUrl = 'https://api.example.com/data';

  // RxJS makes the HTTP request and returns an observable of the response
    return this.http.get(apiUrl);
  }
}
The Power of Angular's RxJS Integration
  • Streamlined Asynchronous Operations
    The observable pattern makes it easy to subscribe to and react to changes in data
    
import { from, Observable } from 'rxjs';

// Create an Observable out of a promise
const data = from(fetch('/api/endpoint'));

// Subscribe to begin listening for async result
data.subscribe({
  next(response) { console.log(response); },
  error(err) { console.error('Error: ' + err); },
  complete() { console.log('Completed'); }
});

Popular RxJS Operators
Map Operator

  • Description: The map operator transforms each item emitted by the source observable by applying a given function
  • Example: Transforming numbers to their squares

import { from } from 'rxjs';
import { map } from 'rxjs/operators';

const numbers = from([1, 2, 3, 4, 5]);
numbers.pipe(
  map(num => num * num)
).subscribe(result => console.log(result)); // Output: 1, 4, 9, 16, 25
Popular RxJS Operators
Filter Operator

  • Description: The filter operator selectively emits items from the source observable that meet a specific condition
  • Example: Filtering even numbers

import { from } from 'rxjs';
import { filter } from 'rxjs/operators';

const numbers = from([1, 2, 3, 4, 5]);
numbers.pipe(
  filter(num => num % 2 === 0)
).subscribe(result => console.log(result)); // Output: 2, 4
Popular RxJS Operators
Merge Operator

  • Description: The merge operator combines multiple observables into one, emitting values from all of them concurrently
  • Merging two data streams

import { merge, of } from 'rxjs';

const source1 = of('A', 'B', 'C');
const source2 = of(1, 2, 3);
const source3 = of(true, false);

merge(source1, source2, source3).subscribe(value => console.log(value));
// Output: 'A', 1, true, 'B', 2, false, 'C', 3
Popular RxJS Operators
SwitchMap Operator

  • Description: The switchMap operator maps each value emitted by the source observable to another observable and cancels any previous inner observable
  • Handling search requests efficiently

import { of } from 'rxjs';
import { switchMap } from 'rxjs/operators';

const source = of(1, 2, 3, 4, 5);

source.pipe(
  switchMap(value => of(`Processed: ${value}`))
).subscribe(result => console.log(result));

// Output:
// Processed: 1
// Processed: 2
// Processed: 3
// Processed: 4
// Processed: 5
